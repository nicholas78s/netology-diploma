 Backend:
 Запускать сервер нужно командой npm run watch, не забудьте сначала сделать npm install, тогда он запустится в режиме live-reload.
 Server:
 Лучше установить nodemon. Делает тоже самое, что и forever, но не лезет в процессы винды.
 npm i nodemon
 В package.json добавить:
 "dev": "nodemon server.js"
 Запуск командой npm run dev
